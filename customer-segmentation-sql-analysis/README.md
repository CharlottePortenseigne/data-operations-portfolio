# Customer Segmentation & Revenue Analysis (SQL)

## Obejctive
Analyze customer purchase behavior to identify high value users and understand revenue distribution.

## Tools
- SQL (SQLite Online)
- CSV dataset

## Key Questions
- Which users spend make the most purchases?
- Which users spend the most?
- What is the average purchase value?
- How can customer be segmented based on spending?
- How is revenue distributed across customer segments?

## Key Analysis
- Purchase frequency by user
- Total spending by user
- Customer segmentation using SQL CASE WHEN

## Conclusion
This analysis highlights differences in customer value and shows that a small number of users can contribute significantly to total revenue.

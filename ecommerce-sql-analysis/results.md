# SQL Results

## Dataset
5 months of e-commerce sample data, with 5,000 rows per month (~25,000 rows total).

## Key Results
- Conversion rate: 3.4%
- Total revenue: 218,830.66
- Most frequent event type: view

## Insights
- The majority of user actions are views
- Purchases represent a much smaller share of activity
- This suggests a strong drop-off between browsing and buying
